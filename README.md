# Campus Navigation System

A Python-based campus navigation system that computes optimal routes between locations using graph algorithms.

## Overview
This project models a campus as a graph and uses Dijkstra’s algorithm to compute optimal paths between locations. It supports both shortest-time routing and accessibility-based routing, allowing users to choose routes based on their needs.

## Features
- Compute shortest path between campus locations  
- Accessibility-aware routing  
- Graph-based campus representation  
- Path reconstruction for route display  
- Simple GUI using Tkinter  

## Technologies Used
- Python  
- Data Structures (Graphs)  
- Algorithms (Dijkstra’s Algorithm)  
- Tkinter  


## Project Structure
- pathFind.py → Graph implementation and Dijkstra’s algorithm  
- main_gui.py → GUI interface  

## How to Run
1. Clone the repository  
2. Navigate to the folder  
3. Run:
   python main_gui.py
